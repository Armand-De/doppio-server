# Doppio Server

## 📚 Description

Essential app for home cafe. Start a home cafe quickly with Doppio.

## 🛠️ Installation

```bash
$ npm install
```

## 🚀 Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 🔒 Environment Configuration


- NODE_ENV


- DB_NAME
- DB_PORT
- DB_HOST
- DB_USERNAME
- DB_PASSWORD


- JWT_ACCESS_TOKEN_SECRET


- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN
- TWILIO_NUMBER
